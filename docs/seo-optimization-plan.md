# SEO Optimization Plan for Homepage

## Current State Analysis

### Issues Identified

1. **Generic Metadata**
   - Root layout uses default "Create Next App" title and description
   - No page-specific metadata for homepage
   - Missing Open Graph tags for social sharing
   - Missing Twitter Card metadata

2. **Missing Semantic HTML Structure**
   - No H1 heading on homepage
   - No descriptive text content explaining what the site is about
   - Content is purely visual (grid of items)

3. **No Structured Data**
   - Missing JSON-LD schema markup (CollectionPage, ItemList, etc.)
   - No breadcrumb schema
   - No organization/person schema

4. **Missing Technical SEO Files**
   - No `robots.txt` file
   - No `sitemap.xml` (Next.js can generate this)
   - No canonical URLs

5. **Image Optimization**
   - Images have alt text but could be more descriptive
   - No lazy loading strategy mentioned
   - Image sizes could be optimized further

6. **Content Gaps**
   - No introductory text explaining the catalogue
   - No keywords naturally integrated
   - Missing context about what visitors will find

## Optimization Recommendations

### Priority 1: Critical SEO Fixes

#### 1. Update Root Layout Metadata
- Replace generic title/description with site-specific metadata
- Add Open Graph tags
- Add Twitter Card metadata
- Set proper canonical URLs

#### 2. Add Page-Specific Metadata
- Create dynamic metadata for homepage
- Include relevant keywords: "design catalogue", "Vitsoe 606", "timeless design", "Dieter Rams inspired"
- Write compelling meta description (150-160 characters)

#### 3. Add Semantic HTML
- Add H1 heading with primary keyword
- Add introductory paragraph explaining the catalogue
- Use proper heading hierarchy

#### 4. Implement Structured Data (JSON-LD)
- CollectionPage schema for homepage
- ItemList schema for the grid of items
- Organization schema for site owner
- BreadcrumbList schema

### Priority 2: Technical SEO

#### 5. Create robots.txt
- Allow all crawlers
- Reference sitemap location

#### 6. Generate Sitemap
- Use Next.js sitemap generation
- Include all pages (homepage, about, item pages)

#### 7. Add Canonical URLs
- Ensure each page has canonical tag
- Prevent duplicate content issues

### Priority 3: Content & UX Enhancements

#### 8. Add Hero Section
- Brief introduction to the catalogue
- Natural keyword integration
- Call-to-action or explanation of purpose

#### 9. Optimize Image Alt Text
- More descriptive alt text
- Include relevant keywords naturally
- Describe what makes each item special

#### 10. Add Footer Content
- Additional navigation links
- Site description
- Keywords in footer (if natural)
